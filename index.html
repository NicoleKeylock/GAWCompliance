🇬🇧
United Kingdom
Post-Brexit. Governed by the FSA and Wine Regulations 2011 as amended. FBO/importer labelling changes from 1 January 2024.
📋 FSA Wine Labelling🏛️ GOV.UK Wine Import🍺 Portman Group
📋 Mandatory Elements
Product Category
Mandatory
Omit if PDO/PGI designation shown (e.g. Rioja satisfies this). Common misunderstanding.
GAW standard: RED WINE / WHITE WINE / ROSE WINE (all caps)
Country of Origin
Mandatory
Always mandatory. Blended wines from multiple countries — all must be named.
GAW standard: PRODUCT OF [COUNTRY]
Net Volume
Mandatory
ml and cl both legally acceptable. Flag cl on own brand as advisory for consistency.
GAW standard: 750ml (lowercase ml, no space)
ABV Declaration
Mandatory
Tolerance ±0.5% up to 5.5% ABV, ±1% above. Must match analysis.
GAW standard: 13% vol. (space between % and vol, full stop)
FBO / Distributor
Mandatory
UK/CI/IoM based business name and address. Not required on retailer own-labels.
GAW standard: DISTRIBUTED BY GAW TA14 6QR, UK
Error to flag: SELECTED BY — must be corrected
Lot Number
Mandatory
L-prefix Julian format. GAW preference: on both bottle neck and label.
Format: L + 2-digit year + 3-digit day + batch e.g. L18065D01
⚠️ Allergens
Sulphites
Mandatory
If present above 10mg/L — applies to virtually all wines.
GAW UK format: Allergy advice: Contains Sulphites
Allergy advice: bold — Contains: regular — allergen: bold
Egg & Milk (fining agents)
If present
Albumin (egg) and casein (milk) declared if residues exceed 0.25mg/L. Confirm via spec.
Example: Allergy advice: Contains Sulphites, Egg
♻️ OPRL Recycling
OPRL Statement
Mandatory
Must accurately reflect the closure type. Most common error: generic Recycle used instead of specific cap instruction.
Screwcap: CAP ON or CAP OFF — Recycle (confirm which applies)
Common error: Label says Recycle but closure needs CAP OFF — flag as critical
Glass Bottle
Mandatory
Glass bottle recycling statement must be present and accurate.
Standard: Widely Recycled — Glass Bottle
🍺 Responsible Drinking
Pregnant Woman Symbol
Mandatory
Crossed-out pregnant woman symbol must appear on all UK wine labels.
Units of Alcohol
Best Practice
Formula: ABV% x Volume(ml) / 1000. Rounded to 1 decimal place.
Example: 13 x 750 / 1000 = 9.8 units
Drinkaware Statement
Best Practice
Requires active GAW Drinkaware licence. Ireland URL only when Republic of Ireland is target market.
PLEASE DRINK RESPONSIBLY
bedrinkaware.co.uk / drinkaware.ie (IE only)
CMO Guidelines
Best Practice
Chief Medical Officers low risk drinking guidelines. Wording is prescribed.
"The UK Chief Medical Officers recommend adults do not regularly drink more than 14 units a week."
📝 GAW Internal Notes
✏️ UK Notes
• SELECTED BY is a legacy error — always correct to DISTRIBUTED BY GAW TA14 6QR, UK • Retailer own-labels do not carry GAW distributor line • Confirm GAW Drinkaware licence is active before any UK label goes to print • PDO/PGI designation satisfies wine category requirement • OPRL statement must match the closure type on the spec — CAP ON/OFF confusion is common • QR code increasingly retailer-expected even though not yet legally required
🇪🇺
European Union
Regulation (EU) 2021/2117. Mandatory nutrition and ingredient labelling from December 2023 for 2024 harvest wines onwards.
⚖️ Regulation 2021/2117🇪🇺 EC Wine Labelling
📋 Mandatory Elements
Energy Declaration
Mandatory
Must be on physical label. Per 100ml in kJ and kcal. Whole numbers only.
GAW standard: E (100ml) : 277kJ / 66kcal
E bold — kJ (lowercase k, uppercase J) — kcal all lowercase
INGREDIENTS & NUTRITION Block
Mandatory
GAW uses Bottlebooks. Unique QR per SKU. New QR required per vintage — never reuse.
INGREDIENTS & NUTRITION
[QR code — min 13x13mm recommended]
E (100ml) : XkJ / Xkcal
Multi-Language Allergen String
Mandatory
Must appear on physical label in languages of all destination markets.
CONTIENT DES SULFITES / CONTAINS SULPHITES / BEVAT SULFIETEN / ENTHALT SULFITE / INDEHOLDER SULFITTER / CONTIENE SULFITOS
FR / EN / NL / DE / DA / ES
Sweetness Term (Sparkling)
Sparkling only
Mandatory for ALL EU sparkling — not just Champagne. Applies to Cremant, Cava, Prosecco, Sekt.
Brut Nature / Extra Brut / Brut / Extra Dry / Dry / Demi-Sec / Doux
Pregnant Woman Symbol
Mandatory
Mandatory on all EU wine labels.
Minimum Font Size
Mandatory
All mandatory information minimum 1.2mm x-height.
Print reminder: Confirm with artwork supplier before print approval
📝 GAW Internal Notes
✏️ EU Notes
• New vintage = new Bottlebooks QR code entry required — never reuse previous vintage QR • Energy declaration (kJ/kcal) must be verified as updated for new vintage — calories change with ABV/sugar • PDO/PGI designation satisfies wine category requirement • Post-Brexit: common back label now achievable for UK+EU due to FBO labelling change • Italy: flag mandatory recycling icons separately when Italy is a target market

Aldi UK + Ireland
Retailer own-label. All design files come from Aldi's team. Dimensions, Pantones, paper, varnish and style guide are specified by Aldi in their design brief. Stage 1: check compliance. Stage 2: proof vs approved artwork.
🏷 Fixed Requirements (every Aldi label)
Volume Format
Aldi standard is 75cl — do NOT flag as non-GAW-standard.
Required: 75cl (not 750ml)
Nutrition — Per 100ml + Per 125ml Glass
Both per 100ml AND per 125ml glass required on Aldi labels.
Example: Per 100ml: 319kJ/76kcal · Per glass (125ml): 399kJ/96kcal
Servings Declaration
Number of servings per bottle must be declared.
Standard: This bottle contains 6 servings
Style Guide Scale
Light Bodied to Full Bodied scale required with position marker. Comes from Aldi design file and spec.
Flavours & Pair With
Flavour descriptor and food pairing sections required on all Aldi back labels.
CMO Guidelines
Full prescribed CMO wording required on all Aldi labels.
"The UK Chief Medical Officers recommend adults do not regularly drink more than 14 units per week."
Drinkaware — Both URLs
Both .co.uk and .ie URLs required on all Aldi UK+Ireland shared labels.
PLEASE DRINK RESPONSIBLY
bedrinkaware.co.uk + drinkaware.ie
Satisfaction Guarantee
Fixed Aldi wording — must match exactly.
"If you are not entirely satisfied with this product please return it to the store where it was purchased and we will be pleased to replace or refund it. This does not affect your statutory rights."
SPECIALLY PRODUCED FOR
Fixed wording with both Aldi addresses. Never changes.
SPECIALLY PRODUCED FOR: Aldi Stores Ltd, PO Box 26, Atherstone, Warwickshire CV9 2SH / Aldi Stores (Ireland) Ltd, PO Box 726, Naas, Co. Kildare. www.aldi.ie
QR Code — Aldi Hosted
Aldi supply and host their own QR code. Not Bottlebooks.
Required wording: Scan for Ingredients and Nutrition
On-Pack Recycle Logo
OPRL logo must be present and accurate for closure type.
Common error: Generic Recycle used when closure needs CAP OFF — flag critical
Aldi Barcode
Barcode is supplied by Aldi — verify it matches what Aldi provided.
Bottled By Statement
Producer name and address required. Aldi format not GAW distributor line.
Allergy Advice Format
Must follow GAW UK standard bold format.
Allergy advice: Contains Sulphites
Vegan Logo
Only include if wine is certified vegan per spec. Flag if present on non-vegan or missing from vegan wine.
Lot Code on Label
Lot code visible on Aldi labels. Julian format preferred.
📝 Notes
✏️ Aldi Notes
• Dimensions, Pantones, paper and varnish all come from Aldi design brief • Aldi supply and host their own QR code — do not set up Bottlebooks for Aldi • FOP (Baron Amarillo example): 118x74mm, Manter Sorolla, Pantone Black 6U/122U/3546U, Estampacion Kurz 457 foil, Barniz Antirayaduras, position 40mm from base • BOP (Baron Amarillo example): 95x75mm, Sorolla H+O Ultra WS, Pantone 440U, Barniz Antirayaduras, position 58mm from base • Printer: Grupo Argraf, Oyon, Alava, Spain (Baron Amarillo — may vary) • Aldi Ireland and UK share the same label — both Drinkaware URLs always required
